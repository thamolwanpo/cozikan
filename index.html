How to use Cozikan

A cozy kanban board built for indie game devs. Here's a walkthrough of all the features using a made-up platformer project.

📌 Cards move between columns — drag a card from one column to another as your task progresses, or click to open it and change the status.
🏷️ Tags keep things organised — use Art, Code, Audio, Design, and Writing to filter visually at a glance. One card can have multiple tags.
🔖 Milestones = sprints — create milestones like "Alpha Build" or "Demo Build" and assign tasks to them. Use the header filter to focus on one milestone at a time.
🔴 Priority bar — the thin coloured stripe on the left of each card shows priority. Red = high, amber = medium, green = low.
